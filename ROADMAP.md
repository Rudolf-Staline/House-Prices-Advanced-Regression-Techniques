# 90-Day Engineering Profile Roadmap

This roadmap organizes practical work across machine learning, software engineering habits, reading, English practice, and written reporting.

## Week 1 — Foundation and first ML baseline

### Goals

- Set up a clean GitHub-ready repository.
- Build the first Kaggle House Prices baseline project.
- Read and summarize initial chapters of *Animal Farm*.
- Start documenting daily and weekly learnings.

### Tasks

#### Repository setup

- Create a professional repository structure.
- Add project, reading, language, and reports folders.
- Add a Python-oriented `.gitignore`.
- Document setup and run commands.

#### Kaggle House Prices baseline

- Prepare a reproducible project under `projects/kaggle-house-prices/`.
- Add data loading and validation utilities.
- Build preprocessing for numeric and categorical features.
- Train Ridge and RandomForest baselines.
- Evaluate models with cross-validation on `log1p(SalePrice)`.
- Generate a Kaggle-compatible submission file.
- Write a short baseline report.

#### Reading — *Animal Farm*

- Read the first part of the book.
- Write a short summary in English.
- Track new vocabulary and useful expressions.

#### Learning documentation

- Keep concise notes about what was built.
- Record blockers, decisions, and next improvements.
- Prepare a short weekly progress report.

## Week 1 deliverables

- Clean repository structure.
- Kaggle House Prices baseline project.
- First EDA notebook.
- Baseline report.
- Initial reading and learning notes placeholders.
